## How to use this plan

This file persists across Claude Code sessions. Each phase should be sized to fit one focused session (~2-4 hours of real work). Claude updates the status of tasks as work progresses. If a phase needs more than one session, split it and update this file.

When starting a new session:
1. Read this file
2. Identify the current phase and first incomplete task
3. Execute
4. Update status here before ending the session

When scope changes mid-phase:
- Never silently drift. Explicitly update PLAN.md with the change and the reason.
- If a task reveals a larger problem, stop and escalate to the human PM.
